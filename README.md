# healthhub-flutter
A comprehensive health &amp; wellness Flutter app — online pharmacy, pet vaccination, blood donation reservations, health events, and a medical blog. Built with Firebase &amp; FlutterFlow.

## Features

- **Authentication** — Email/password signup, Google Sign-In, and password reset
- **Online Pharmacy** — Browse and order medications directly from the app
- **Pet Vaccination** — Schedule and track pet vaccination appointments
- **Blood Donation** — Find donation centers and reserve a donation slot
- **Health Events** — Discover and register for local health events
- **Health Blog** — Read and explore curated medical and wellness articles

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Flutter 3.32 |
| Builder | FlutterFlow v6.3 |
| Backend | Firebase (Auth, Firestore) |
| Platform | iOS & Android |
| Auth | Firebase Auth + Google Sign-In |

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/YOUR_USERNAME/healthhub-flutter.git
```
2. Install dependencies
```bash
   flutter pub get
```
3. Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) from your Firebase console
4. Run the app
```bash
   flutter run
```
